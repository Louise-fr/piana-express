ASTUCES :
	HTML
	- appliquer "100vh" pour height et weight pour ne plus s'inquiéter de l'héritage depuis <html> ou <body>

	GIT :
	git remote add origin git@github.com:Louise-fr/piana-express.git pour lier au repo en ligne
	git add .
	git reset . (si fichier ajouté par erreur)
	git config --global core.safecrlf false (pour désactiver warning crlf)
	git commit -m "initial commit"
	git push -u origin master

	Run
  	-git config --global user.email "you@example.com"
  	-git config --global user.name "Your Name"
	to set your account's default identity.
	Omit --global to set the identity only in this repository.
