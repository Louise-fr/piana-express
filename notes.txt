Vue CLI est basé sur : webpack-dev-server
	Use webpack with a development server that provides live reloading. This should be used for development only.
	It uses webpack-dev-middleware under the hood, which provides fast in-memory access to the webpack assets.


Coté client :
	-NPM : node package manager
	dans /client : npm run serve

	- browserify permet la gestion des js personalisés + css 
	(modifier entry.js puis "browserify entry.js --debug > bundle.js")

Côté serveur API :
	Express JS pour les routes
	GAPI pour gestion des emails

	GIT :
	git remote add origin git@github.com:Louise-fr/piana-express.git pour lier au repo en ligne
	git add .
	git reset . (si fichier ajouté par erreur)
	git config --global core.safecrlf false (pour désactiver warning crlf)
	git commit -m "initial commit"
	git push -u origin master

	Run
  	-git config --global user.email "you@example.com"
  	-git config --global user.name "Your Name"
	to set your account's default identity.
	Omit --global to set the identity only in this repository.
